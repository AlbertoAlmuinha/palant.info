We’ve hidden `jQuery.buildFragment` in jQuery 3.0.0 (not released yet but there’s an alpha to try) so this technique won’t work for you. But at the same time we’ve exposed `jQuery.htmlPrefilter` that’s invoked on the HTML input any time jQuery tries to parse something less trivial than a single tag without content so you won’t have to rely on private APIs, you’ll just be able to:

jQuery.htmlPrefilter = function () { return ‘’; }

and then:

jQuery(’<div>’); // [<div></div>]

jQuery(’<div></div>’); // [<div></div>]

jQuery(’<div>foo</div>’); // []

Please report bugs if you find that it’s insufficient for some reason!
